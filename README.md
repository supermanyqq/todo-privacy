
# 隐私政策

Todo 是一个用于快速创建待办事项的App。


#### Todo 收集的数据

简而言之，Todo 应用程序本身不会收集您的任何数据。

但是您应该谨慎地对待一些可能会请求您数据的 API，例如位置、日历和提醒。

您所有隐私信息永远保留在您的设备上，我们无法控制它。

#### 本隐私政策的变更

我们可能会不时地更新我们的隐私政策。因此，建议您定期查看此页面以了解更新后的内容。我们会通过在此页面上发布新的隐私政策来通知您任何更改。这些更改在此页面上发布后立即生效。
